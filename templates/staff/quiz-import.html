JSON Structure
Quiz Fields
  • title - Quiz title (required)
  • description - Quiz description
  • difficulty - beginner, intermediate, advanced, expert
  • time_limit - Time in seconds (default: 600)
  • pass_percentage - Pass threshold (default: 70)
  • xp_reward - XP reward (default: 10)
  • is_published - true/false
  • is_featured - true/false
Question Fields
  • text - Question text (required)
  • question_type - single, multiple, true_false, code
  • code_snippet - Code to display
  • code_language - python, javascript, etc.
  • explanation - Answer explanation
  • points - Points (default: 1)
Option Fields
  • text - Option text (required)
  • is_correct - true/false
Tips
  • Category is selected separately (not in JSON)
  • Each question must have at least one correct option
  • Use is_published: true to publish immediately
  • Slug is auto-generated from title
  • Quiz is created by current staff user
