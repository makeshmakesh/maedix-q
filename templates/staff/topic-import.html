JSON Structure
Topic Fields
  • title - Topic title (required)
  • description - Topic description
  • status - draft, published
  • is_featured - true/false
  • estimated_time - Minutes (default: 2)
  • thumbnail_url - S3 URL for thumbnail
Card Fields
  • title - Card title (optional)
  • content - Card content (required)
  • card_type - text, text_code, text_image
  • code_snippet - Code for text_code type
  • code_language - python, javascript, etc.
  • image_url - S3 URL for text_image type
  • image_caption - Image caption
Tips
  • Category is selected separately (not in JSON)
  • Keep card content to ~100 words for best readability
  • Use status: "published" to publish immediately
  • Slug is auto-generated from title
  • Cards are ordered as they appear in the JSON array
  • Use text_code type to include code snippets
  • Use text_image type to include images
Card Types
  • text - Text only card
  • text_code - Text with code snippet
  • text_image - Text with image
